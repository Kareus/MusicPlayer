음악 탭을 제외한 모든 탭은 이름 검색만 지원합니다.

음악 탭에서 지원하는 검색 필터는 다음과 같습니다.
name: (이름. 필터명이 누락된 경우 자동으로 이 필터로 인식합니다. 다른 탭도 동일합니다)
artist: (아티스트)
album: (앨범)
genre: (장르)
folder: (폴더)
composer: (작곡가)
writer: (작사가)
lyrics: (가사)

특수 필터
recentadd: (최근 추가)
mostplay: (가장 많이 재생)

-------------------
특수 필터는 다른 필터보다 우선 인식되고, 이 경우 다른 필터는 모두 무시됩니다. (중복 적용 불가)
필터를 여러 개 적용할 경우, |를 이용해 필터를 구분합니다.

필터 검색은 현재 목록에 대해 검색합니다. 다시 말해, 검색할 때마다 필터가 누적 적용됩니다.
사용 예시는 다음과 같습니다.

Offical (태그 누락. name으로 인식하여 이름에 Offical이 들어가는 데이터 검색)
name:Official|artist:10cm (이름에 Official이 들어가고 아티스트명에 10cm가 들어가는 데이터 검색)
